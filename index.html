Projetos

Quiz em Java

✓ Novembro de 2025

Projeto criado como atividade avaliativa para a matéria de Programação Orientada a Objeto (POO) na faculdade.

Programa simples feito na linguagem Java (versão 25) que executa um quiz no terminal visando aplicar conceitos de POO como encapsulamento, herança, polimorfismo, sobrescrita, sobrecarga, entre outros. Esse sistema foi feito com espaço para aprimoramento, podendo futuramente ter, por exemplo, a opção do usuário escolher entre múltiplas instâncias de Quiz disponíveis.

AppClimaBom

(T.B.D) Novembro de 2025

Site em desenvolvimento que permite ao usuário consultar o tempo da localidade que inserir, exibindo previsões meteorológicas diárias de até 1 semana. No futuro próximo, exibirá também alertas meteorológicos e previsões da mudança de clima ao longo do dia atual.

A linguagem Python está sendo utilizada por todo projeto, com o uso das bibliotecas Streamlit para a geração da interface web e GeoPy para conversão de strings (localidades) em coordenadas.
